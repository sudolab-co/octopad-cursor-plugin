# Octopad

The back-office workspace for your team's AIs: shared tasks, knowledge, and context over MCP.

Octopad is a **remote MCP server**. This plugin registers the hosted endpoint
(`https://mcp.octopad.app/mcp`); Cursor handles authentication via OAuth 2.0 with
PKCE on first connect. No local installation or API keys required.

See the [repository README](../../README.md) for details.
